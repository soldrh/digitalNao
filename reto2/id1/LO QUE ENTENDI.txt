ES UNA FORMA DE CONSULTAR SCOPUS DE FORMA AUTOMATIZADA SIN INTERACCION HUMANA.
LAS APIS OFRECEN LAS MISMAS FUNCIONALIDADES DE LA INTERFAZ DE USUARIO DE SCOPUS Y 
PERMITEN QUE LOS PROGRAMAS CONSULTEN, ENCUENTREN Y DEVUELVAN LOS DATOS EN SCOPUS.

SON IMPORTANTES PORQUE SON UNA BASE DE DATOS.

SCOPUS SEARCH API ES UNA INTERFAZ DE PROGRAMACION DE APLICACIONES (API) PROPORCIONADA 
POR ELSEVIER QUE PERMITE A LOS USUARIOS REALIZAR BUSQUEDAS AVANZADAS EN LA BASE DE DATOS 
DE SCOPUS, QUE CONTIENE MAS DE 80 MILLONES DE REGISTROS DE ARTICULOS DE REVISTAS, LIBROS, 
CONFERENCIAS, PATENTES Y OTROS TIPOS DE PUBLICACIONES CIENTIFICAS Y TECNICAS.

LAS CUOTAS Y TASAS DE LIMITACION SE PUEDEN VER SON UNICOS PARA CADA API, NO HAY UNA SOLA
CONFIGURACION GLOBAL PARA UN APIKEY DETERMINADO. SI UNA CU0TA NO ES SUFICIENTE SE PUEDE
ESCRIBIR AL SOPORTE DE API CON SU CLAVE DE API.

UNA API KEY SIRVE PARA USAR EL SERVICIO DE LA INTERFAZ DE PROGRAMACION DE APLICACIONES.
